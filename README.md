# Frontend Start Card

Мой первый учебный мини-проект по фронтенду.

## Что внутри
- HTML
- CSS
- JavaScript
- SVG-логотип

## Что умеет страница
- Показывает простую карточку
- Подключает стили, скрипт и изображение
- Меняет заголовок по клику на кнопку

## Как запустить локально
Открыть проект через Live Server в VS Code.

## Онлайн-версия
Ссылка появится после публикации через GitHub Pages.

## Мои цели по проекту
- Научиться работать с Git и GitHub
- Сделать первый публичный репозиторий
- Подготовить базу под портфолио
